# Operational Research 

Console-based application that implements classic Operational Research algorithms and optimization methods in Python.

## Implemented Algorithms

- Transportation Problem — Stepping Stone Method
- Minimum Spanning Tree — Kruskal
- Floyd–Warshall Algorithm

## Console Interface

Example menu:

```text
[1] Transporte - Stepping Stone
[2] MST - Kruskal
[3] Floyd-Warshall
[4] Salir
```

The program allows the user to:
- Use default examples
- Enter custom graphs or matrices
- Solve optimization problems directly from the console

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/Victor-Rojas-09/Operational_Research.git
cd Operational_Research
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the project:

```bash
python -m Operational_Research
```
